# Lenguajes de Marcas — ASIR

Repositorio de apoyo para el módulo **Lenguajes de Marcas** del ciclo formativo de **Administración de Sistemas Informáticos en Red (ASIR)**.

Este repositorio contiene:

- Ejemplos utilizados en clase
- Datasets XML empleados durante el módulo
- Retos guiados
- Soluciones de referencia
- Materiales complementarios

## Estructura

/datasets → documentos XML utilizados durante el curso  
/ejemplos → ejemplos vistos en clase  
/retos → ejercicios propuestos al alumnado  
/soluciones → soluciones de los retos  

## Tecnologías trabajadas

- XML
- XSD
- XSLT
- XPath
- XQuery
- HTML
- CSS
